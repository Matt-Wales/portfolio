About Me

I'm a baseball aficionado with a passion for using software to automate baseball operations workflows and find solutions to baseball problems. I graduated from the University of Minnesota in December 2018 with a bachelors degree in economics and a minor in statistics.

I started my professional baseball career in the 2019 season when I joined the minor league video department of the Arizona Diamondbacks.

My first programming experience was basic statistical analysis with R in college, and I started making mobile apps and web apps in early 2020.

My Projects

Advance Scouting Mobile App

Flutter Dart Provider SQLite MiLB.com API Firebase

A fully-fledged cross-platform (iOS and Android) mobile app that completely streamlines and automates advance scouting report generation at the minor league level. The app's auxiliary features include a lineup builder that allows a coach to enter the opposing team's batting lineup for the day, as well as post-game breakdowns automatically generated from TrackMan data that display pitch locations and details.

MiLB Affiliates Tracker

R Plotly

A simple R application that illustrates the profound geographical and organizational changes, with regards to how each Major League club's affiliates are structured, that have occurred in Minor League Baseball between 2019 and 2021. Built with the Plotly library. Some league alignments are speculatory due to official league reassignments at the High-A level not being released yet.

Opposing Hitter and Pitcher Dashboards

R Markdown Python Flexdashboard

A suite of HTML dashboards written in R Markdown that display basic advance scouting data for a given opposing team's hitters and pitchers. Browse each hitter's metrics compared to league averages and view their heatmaps according to pitcher handedness and pitch type. Browse each pitcher's movement metrics compared to league averages, velocity charts, pitch type arsenals, and video.
